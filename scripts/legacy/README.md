# Legacy benchmark scripts

The scripts in this subdirectory are kept only for historical reference.
All benchmark operations are handled by `tools/bench.py`.

## Current benchmark entry points

| Purpose | Command |
|---|---|
| Local corpus benchmark | `tools/bench.py local` |
| Render local results | `tools/bench.py render --view local` |
| Full scoreboard refresh (Ganak + native) | `tools/bench.py full` |
| Render full scoreboard | `tools/bench.py render --view full` |
